### Current Implementation (index-clarity.html)

**User Flow:**
1. Click "Add Validation" button
2. Modal opens with list of all 21 validation types
3. Select one or more types (multi-select)
4. Choose severity (ERROR/WARNING)
5. Click "Add Validation"
6. Fill in parameters in the main form

**Pros:** Simple, all options visible
**Cons:** Overwhelming with 21 types, no search, no categorization in modal

## â­ Recommended Approach

**Hybrid: Quick Picker + Detailed Form**

**Why this approach:**

1. **Fast for power users:** Type to search, keyboard navigate
2. **Discoverable for new users:** See all options, read descriptions
3. **Contextual:** Shows only relevant validations for selected file type
4. **Progressive disclosure:** Simple picker â†’ detailed form â†’ advanced options
5. **Familiar:** Matches VS Code, IntelliJ, Sublime UX patterns

**Implementation:**

```javascript
// Quick Picker Component
class ValidationQuickPicker {
  constructor() {
    this.validationTypes = VALIDATION_TYPES; // From APP_CONFIG
    this.filtered = this.validationTypes;
  }

  show(fileType, existingValidations) {
    // Filter out already-added validations
    this.filtered = this.validationTypes.filter(
      v => !existingValidations.includes(v.type)
    );

    // Create picker UI
    const picker = document.createElement('div');
    picker.className = 'quick-picker';
    picker.innerHTML = `
      <input type="text"
             class="quick-picker-search"
             placeholder="ðŸ” Search validations..."
             autofocus>
      <div class="quick-picker-results"></div>
    `;

    // Attach to visual editor panel
    document.getElementById('visual-editor').prepend(picker);

    // Handle search
    const searchInput = picker.querySelector('.quick-picker-search');
    searchInput.addEventListener('input', (e) => {
      this.filterResults(e.target.value);
    });

    // Handle selection
    picker.addEventListener('click', (e) => {
      if (e.target.closest('.validation-item')) {
        const type = e.target.closest('.validation-item').dataset.type;
        this.selectValidation(type);
      }
    });
  }

  filterResults(query) {
    const results = this.filtered.filter(v =>
      v.name.toLowerCase().includes(query.toLowerCase()) ||
      v.description.toLowerCase().includes(query.toLowerCase())
    );

    this.renderResults(results);
  }

  renderResults(results) {
    const container = document.querySelector('.quick-picker-results');
    container.innerHTML = results.map(v => `
      <div class="validation-item" data-type="${v.type}">
        <span class="validation-icon">${v.icon}</span>
        <div class="validation-info">
          <strong>${v.name}</strong>
          <p>${v.description}</p>
        </div>
      </div>
    `).join('');
  }

  selectValidation(type) {
    // Close picker
    document.querySelector('.quick-picker').remove();

    // Open parameter form
    ValidationForm.show(type);
  }
}
```
